# research/

Open notes: comparisons between agents/models, evaluations, experiments,
and links worth keeping — things being explored that haven't (yet)
crystallized into a reproducible entry under `setups/`.

Add a new note with [`../templates/research-template.md`](../templates/research-template.md),
named `research/<slug>.md`. When a note concludes into a real setup, link
forward to it from the note and mark the note's status `concluded`.
