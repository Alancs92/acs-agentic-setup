# templates/

Copy-paste starting points for adding new content, so entries stay
structurally consistent enough that an agent (or me, months later) can scan
just the metadata and headings without reading the full file.

| Template | Use for |
|---|---|
| [`agent-template.md`](agent-template.md) | A new file under `agents/<name>/README.md` — documenting an agent tool itself. |
| [`setup-template.md`](setup-template.md) | A new `setups/<slug>/README.md` — a concrete, reproducible setup. |
| [`research-template.md`](research-template.md) | A new note under `research/` — an open-ended exploration or comparison. |

Copy the file, fill in the metadata block at the top first, then the body.
